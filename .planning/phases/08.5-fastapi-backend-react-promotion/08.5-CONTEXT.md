<code_context>
## Existing Code Insights

### Reusable Assets
- `src/matching/engine.py` — `compute_match_score()`, `rank_speakers_for_event()`, `generate_pipeline_data()`
- `src/matching/factors.py` — 6 pure scoring functions (topic_relevance, role_fit, geographic_proximity, calendar_fit, historical_conversion, student_interest)
- `src/ui/data_helpers.py` — Cached loaders: `load_specialists()`, `load_cpp_events()`, `load_pipeline_data()`, `load_event_calendar()`, `load_poc_contacts()`
- `src/outreach/email_gen.py` — `generate_outreach_email()` (Gemini-powered with fallback)
- `src/outreach/ics_generator.py` — `generate_ics()` (pure RFC 5545 string building)
- `src/ui/outreach_bridge.py` — `build_outreach_params()` (pure transform, no Streamlit deps)
- `src/config.py` — `FACTOR_REGISTRY`, `DEFAULT_WEIGHTS`, `METRO_REGIONS`, all constants

### Established Patterns
- Pure functions with full type annotations
- Immutable configs (frozen dataclasses, Final types)
- Graceful degradation (fallbacks for missing data, LLM errors)
- Cached data loaders via `@functools.lru_cache`
- Data files in `data/` directory (CSV + JSON)

### Integration Points
- V1.1 React mockup at `docs/mockup/V1.1/IA-West_UI/` — React 18 + Vite 6 + Tailwind v4 + shadcn/ui + Recharts + react-router v7
- 7 pages already scaffolded: Dashboard, AIMatching, Calendar, Opportunities, Outreach, Pipeline, Volunteers
- 50+ shadcn/ui components in `src/app/components/ui/`
- Layout component with navigation at `src/app/components/Layout.tsx`

</code_context>
